# ToDo PM - 思维导图 + 任务管理工具

一个网页版的项目推进管理工具，深度融合 Xmind 风格思维导图与滴答清单式任务管理，实现思维导图节点与任务的无缝双向转化。

## 核心理念

在思考与规划结构（思维导图）的同时，可直接将任意节点转为可执行任务，进行进度跟踪、时间管理和协作。

## 技术栈

- **前端框架**：Vue 3 + TypeScript + Vite
- **状态管理**：Pinia
- **UI 组件库**：Element Plus
- **思维导图**：SimpleMindMap（[wanglin2/mind-map](https://github.com/wanglin2/mind-map)）
- **持久化**：IndexedDB + LocalStorage
- **E2E 测试**：Playwright

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 运行 E2E 测试
npx playwright install chromium  # 首次需要安装浏览器
npx playwright test               # 运行所有测试
npx playwright test --ui          # 交互模式运行测试
```

开发服务器默认运行在 `http://localhost:5174`。

## 核心功能

### 1. 完整思维导图功能
- 基于 SimpleMindMap 集成，支持节点增删改、富文本编辑
- 支持多种布局切换（逻辑结构、思维导图、目录组织、组织结构等）
- 支持主题切换（默认、经典、深色、简洁等）
- 支持撤销重做、快捷键操作
- 数据自动持久化到 IndexedDB

### 2. 节点转任务系统（核心创新点）
- 选中任意节点，点击工具栏"转为任务"或右键菜单操作
- 任务属性：标题、描述、截止日期、优先级（高/中/低）、标签、负责人、进度
- 任务状态：未开始 / 进行中 / 已完成
- 任务节点在思维导图中有明显视觉区分（优先级对应颜色边框）
- 支持将任务节点转回普通节点

### 3. 任务管理功能
- 独立任务列表视图，参考滴答清单风格
- 五种筛选视图：今日、接下来、本周、全部、已完成
- 搜索、排序（按创建时间、截止日期、优先级、状态）
- 任务详情面板（侧滑抽屉）
- 子任务支持（添加/删除子任务）
- 父任务进度自动计算

### 4. 双向同步机制
- 思维导图节点标题修改 → 任务标题自动同步
- 任务列表修改状态/优先级 → 导图节点视觉样式实时更新
- 任务列表修改标题 → 导图节点文字实时更新
- 回调模式避免循环同步

### 5. 数据持久化
- IndexedDB 自动保存思维导图数据和任务数据
- 刷新页面后数据完整恢复
- 页面关闭重开后数据不丢失

## 项目结构

```
src/
├── App.vue                      # 根组件（顶部导航 + 路由视图）
├── main.ts                      # 应用入口
├── style.css                    # 全局样式（深色主题）
├── env.d.ts                     # 类型声明
├── router/
│   └── index.ts                 # 路由配置（/mindmap, /tasks, 404 catch-all）
├── stores/
│   ├── task.ts                  # 任务状态管理（Pinia）
│   └── mindmap.ts               # 思维导图状态管理（Pinia）
├── types/
│   ├── task.ts                  # 任务类型定义
│   └── mindmap.ts               # 思维导图类型定义
├── utils/
│   └── db.ts                    # IndexedDB 持久化层
├── views/
│   ├── MindMapView.vue          # 思维导图视图
│   └── TaskListView.vue         # 任务列表视图
└── components/
    └── task/
        └── TaskDetailPanel.vue  # 任务详情面板
```
