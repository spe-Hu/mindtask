# MindTask 更新日志

本文件记录 MindTask 项目的所有重要变更。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-06-09

### 新增
- **完整国际化支持**：中英文双语界面，182+ 条翻译词条
- **主题系统**：7 种预设主题（默认、经典、深色、简洁、多彩、森林、日落）
- **快捷键系统**：
  - `?` 显示快捷键帮助面板
  - 方向键在思维导图节点间导航
  - `Q` 快速添加任务
  - `Ctrl/Cmd+F` 全局搜索
  - 完整的快捷键文档
- **多视图支持**：
  - 思维导图视图（核心功能）
  - 任务列表视图
  - 看板视图
  - 日历视图
  - 统计仪表板
- **Electron 桌面应用**：
  - macOS / Windows / Linux 跨平台支持
  - 原生窗口控制
  - 本地数据存储
- **增强功能**：
  - 全局搜索组件
  - 快速添加任务面板
  - Markdown 编辑器
  - 任务详情面板
  - 时间追踪
  - 子任务支持
  - 评论系统
  - 活动日志
- **多项目管理**：
  - 项目切换
  - 项目级数据隔离
  - 路由支持 `/project/:projectId/...`

### 改进
- 思维导图与任务双向同步机制
- IndexedDB 数据持久化
- 响应式 UI 设计
- 性能优化（虚拟滚动、懒加载）

### 技术栈
- Vue 3.5 + TypeScript 6.0
- Vite 8.0 构建工具
- Pinia 状态管理
- Element Plus UI 组件库
- SimpleMindMap 思维导图引擎
- Electron 42.3 桌面应用框架
- Playwright E2E 测试

## 版本说明

### 版本号规则
- **主版本号**：不兼容的 API 变更
- **次版本号**：向后兼容的功能新增
- **修订号**：向后兼容的问题修复

### 变更类型
- **新增**：新功能
- **改进**：现有功能的优化
- **修复**：问题修复
- **移除**：移除的功能
- **安全**：安全相关修复

## 升级指南

### 从 0.1.0 升级到 1.0.0

1. **数据兼容性**：1.0.0 完全兼容 0.1.0 的数据格式，无需迁移
2. **新功能启用**：
   - 国际化：默认中文，点击右上角按钮切换语言
   - 主题：点击主题选择器切换
   - 快捷键：按 `?` 查看所有快捷键
3. **注意事项**：
   - 新增多项目支持，旧数据会自动归入"默认项目"
   - 快捷键可能与浏览器冲突，建议使用应用内快捷键

## 已知问题

### 1.0.0 已知问题

1. **Electron 构建超时**：网络原因导致 Electron 二进制下载超时
   - 解决方案：使用国内镜像 `export ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/`

2. **路径中文字符**：项目路径含中文可能导致某些工具出错
   - 解决方案：移动到纯英文路径

3. **macOS 签名**：本地测试已禁用 hardened runtime
   - 正式发布需要配置开发者签名
